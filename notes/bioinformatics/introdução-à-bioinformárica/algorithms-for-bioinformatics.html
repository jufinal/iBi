Algorithms

Definition

An algorithm is a specific set of instructions for carrying out a procedure or solving a problem, usually with the requirement that the procedure terminate at some point. Specific algorithms sometimes also go by the name method, procedure, or technique. The word "algorithm" is a distortion of al-Khwārizmī, a Persian mathematician who wrote an influential treatise about algebraic methods.

Source: Wolfram MathWorld

Accounting Algorithm

  • When dealing with localized features in order to detect local occurrence.
    • When the sought feature is found the (Account) Balance is incremented (Credit)
    • If it is not found he (Account) Balance is decremented (Debit)
    • The site boundary begins when Balance starts growing from a local minimum.
    • The site ends when the Balance stops growing.
    • Balance progression

    • Case Study
      • GpC Island Detection
        • CpG Island
          • What is...

            Involved in resistance to methylation “associated with the 5' ends of most”(mammalian)” house-keeping genes and many regulated genes (Bird, 1986, Larsen et al 1992)”

          • General characteristics
            • > 200 bp
            • %G+C = 50%
        • Example Implementation
          • EMBOSS::cpgreport

Sliding Windows Algorithms

  • The sequence is considered n-odd residues at a time, the so called Window.
  • At each iteration
    • This Window slides one site towards the end of the sequence
      • Sliding along the sequence

    • The properties of the residues inside the Window are considered for calculation purposes
      • This calculation is typically based on a given Metric (scoring scheme).
      • Each calculated value encompasses both the site where the Window is centered into, and its neighbourhood.
